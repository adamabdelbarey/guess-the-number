# 🎯 Guess The Number Game

A simple and interactive **Guess The Number** game built using **HTML, CSS, JavaScript, and Bootstrap 5**.  
The player tries to guess a randomly generated number within a limited number of chances.

## 🧠 Game Concept

- A random number is generated between **1 and 10**
- The player has **3 chances** to guess correctly
- After each guess, the game gives hints:
  - 📈 Higher
  - 📉 Lower
- The game ends when:
  - The player guesses correctly 🎉
  - Or runs out of chances ❌
